# 選定ルール

## 1. 選定基準(基本ルール・固定)

部品選定は、**各メーカーカタログに記載の選定基準・選定表・選定フローチャートに従う**。
このルールはデータ(過去実績等)が蓄積しても変更しない。

例:
- ブレーカー → 負荷電流・短絡容量からの選定表
- 電磁接触器 → 負荷の種類・使用頻度からの選定表
- 電磁開閉器 → 上記に準ずる選定基準

## 3. 過去実績の扱い(参考情報)

- 過去に同条件・類似条件で採用した品番があれば、`index.csv` の `past_adoption`
  に記録し、Agent03の出力時に参考情報として提示する
- 過去実績は選定の決定要因にはしない。あくまでカタログ基準での選定結果と
  一致するかどうかの確認・補足情報として使う

## 4. 選定基準が確認できない場合

- 該当カテゴリ・シリーズの選定基準ページが `index.csv` に未記録、または
  カタログPDF内で見つからない場合、選定を確定せず、その旨をユーザーに報告する
- 確認後、`selection_guide_ref` を更新する

## 5. 保留事項・今後の検討課題

### 富士SC-NEXTの大容量帯対応(2026-06-13・仮説段階)

- SC-NEXTカタログ(電磁接触器、開閉器.pdf)のラインアップ表(p.4-5)では、
  最大フレームSC65X/SW65Xで以下までしかカバーしていない。
  - 200-240V: 最大15kW(65A)
  - 380-440V: 最大30kW(65A)
- 三菱(MS-T/Nシリーズ)はN125〜N800AB(〜200kW級)まで1シリーズでカバーしているのに対し、
  富士はSC-NEXTが中・小容量帯のみで、大容量帯は別シリーズ・別カタログになる可能性が高い。

**対応条件**: 「部品カタログ」フォルダに大容量帯向けの富士カタログが追加された時点で、
- index.csvへの新規行追加
- SC-NEXT側のnotesに「適用容量範囲(上限)」を追記済み(本対応は完了)

→ 大容量カタログ追加後、上記の新規行追加のみ残作業。
