# 適用規格

設備案件で参照する規格一覧。案件ごとに適用要否を確認すること。

---

## 1. 低圧電気機器・盤設計（国内向け基本）

| 規格番号 | 名称 | 主な適用場面 |
|---|---|---|
| JIS C 8201 | 低圧開閉装置及び制御装置 | ブレーカ・電磁接触器・電磁開閉器全般 |
| JIS C 8201-2-1 | 配線用遮断器(MCCB) | ブレーカ選定・定格確認 |
| JIS C 8201-4-1 | 電磁接触器・電磁開閉器 | 電磁接触器・電磁開閉器選定 |
| JEM 1195 | 電磁開閉器 | 国内仕様の電磁開閉器選定基準 |
| JEM 1038 | 配線用遮断器 | 国内仕様のブレーカ選定基準 |
| JIS C 0920 | 外郭保護等級(IP) | 制御盤・機器の保護等級指定 |
| JIS C 3307 | 600V ビニル絶縁電線(IV) | 盤内配線の電線選定 |
| JIS C 3316 | 電気機器用ビニル絶縁電線(KIV) | 盤内制御配線の電線選定 |

## 3. 安全・機械指令（輸出・CE対応が必要な場合）

| 規格番号 | 名称 | 主な適用場面 |
|---|---|---|
| IEC 62061 | 機械類の安全 — 安全関連電気制御システム | 安全回路設計(SIL評価) |
| ISO 13849-1 | 機械類の安全 — 制御システムの安全関連部(PL評価) | 安全回路設計(PL評価) |
| IEC 60947 | 低圧開閉装置及び制御装置(国際規格) | CE対応機器選定 |
| EU 機械指令 2006/42/EC | 機械指令 | EU向け輸出機械 |

> Agent01 が「CE対応」「輸出向け」を確認した場合は、上記安全規格の適用要否をユーザーに確認する。

## 運用ルール

- Agent01 が仕様解析時に「適用規格」欄を確認し、要求がある場合は本ファイルを参照する
- 新規規格が案件で確認された場合は本ファイルに追記する（変更履歴をAgent00に報告）
- 国内向け通常案件では「1. 低圧電気機器・盤設計」が基本適用範囲
