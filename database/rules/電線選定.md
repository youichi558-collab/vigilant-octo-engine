# 電線選定ルール

Agent03が電線サイズを選定する際に参照するルール・計算式・許容電流表。
カタログPDFは使用しない（本ファイルで完結する）。

根拠法令: 電気設備技術基準の解釈 第148条(幹線)・第149条(分岐回路)

---

| 種別 | 規格 | 主な用途 |

| IV（600Vビニル絶縁電線） | JIS C 3307 | 主回路幹線・分岐配線 |
| HIV（600V二種ビニル絶縁電線） | JIS C 3307 | 主回路（高温環境・耐熱が必要な箇所） |

| 周囲温度 | 補正係数 |

> 盤内配線は盤内温度が上昇しやすいため、周囲温度を40℃以上で計算することが多い。

## 4. 電線サイズ選定フロー

```
① 負荷電流 I_L を確認する（spec_summary.md の機器一覧から）

② 補正後の許容電流を計算する
   I_allow = 表の許容電流 × 温度補正係数 × 束線補正係数

③ I_allow ≥ I_L となる最小サイズを許容電流表から選ぶ

④ 電圧降下チェック（下記「5. 電圧降下計算」を参照）
   ΔV ≤ 基準値 を満たすか確認する

⑤ ブレーカとの保護協調チェック（下記「6. 保護協調」を参照）

⑥ ③〜⑤をすべて満たす最小サイズを採用サイズとする
```

## 5. 電圧降下計算

### 計算式

**三相回路（三相モータ等）**
```
ΔV [V] = √3 × I [A] × L [m] × r [Ω/km] / 1000
```

**単相回路（制御回路・照明等）**
```
ΔV [V] = 2 × I [A] × L [m] × r [Ω/km] / 1000
```

### 主要電線の導体抵抗（20℃、銅）

| 公称断面積 | 導体抵抗 r [Ω/km] |
|---|---|
| 2mm² | 9.61 |
| 3.5mm² | 5.54 |
| 5.5mm² | 3.39 |
| 8mm² | 2.30 |
| 14mm² | 1.31 |
| 22mm² | 0.842 |
| 38mm² | 0.482 |
| 60mm² | 0.307 |
| 100mm² | 0.181 |

### 電圧降下の許容値（目安）

| 対象 | 基準 | 根拠 |
|---|---|---|
| 幹線（電源〜主幹ブレーカ） | ΔV ≤ 2% | 電技解釈 第148条 |
| 分岐回路（主幹〜負荷端） | ΔV ≤ 2% | 電技解釈 第149条 |
| 幹線＋分岐の合計 | ΔV ≤ 4% | 電技解釈 第148条ただし書き |

> 例: 200V系で幹線2%なら ΔV ≤ 4V

## 6. ブレーカとの保護協調

電線は過電流からブレーカで保護される必要がある。

| 条件 | 要件 |
|---|---|
| 電線の許容電流 ≥ ブレーカの定格電流 | 原則（電技解釈 第148・149条） |
| 例外（幹線の場合） | 一定条件下でブレーカ定格が許容電流を超えることを許容（電技解釈 第148条 表参照） |

> Agent03がブレーカを選定した後、ブレーカ定格と電線許容電流の整合を必ず確認すること。
> 整合しない場合は電線サイズを上げるか、ブレーカ定格を下げる。

## 7. Agent03での使い方

1. `spec_summary.md` から各回路の負荷電流・線路長・設置環境（盤内温度等）を確認する
2. 本ファイルの「4. 電線サイズ選定フロー」に従って計算する
3. 計算結果・根拠（計算式・使用した補正係数）を `parts_selection.md` に記録する
4. 線路長が不明な場合はユーザーに確認する（電圧降下計算に必須）
5. 選定結果は「候補一覧」ではなく「計算結果に基づく推奨サイズ」として提示する

## 8. 電線色の標準

### 8-1. 国内向け標準（FA設備・制御盤）

参考文献: Techno Wizard 電装盤製作仕様書（2024年5月版）

| 回路種別 | 色 | 備考 |
|---|---|---|
| 主回路 第1相（R/U） | 赤 | |
| 主回路 第2相（S/V） | 白 | |
| 主回路 第3相（T/W） | 黒（または青） | |
| 接地線（E/PE） | 緑 | 電技解釈・内線規程で規定（変更不可） |
| 交流制御回路（共通母線・中間線） | 黄 | L/N/R/S いずれも黄 |
| 直流制御回路（共通母線P/N） | 青 | ＋/－ともに青。極性は端子番号・回路番号で区別 |
| センサ配線 3線式（＋/信号/0V） | 茶/黒/青 | センサ標準色（IEC準拠） |
| センサ配線 2線式（信号/0V） | 茶/青 | 同上 |

> **PE（緑）のみ法的根拠あり。それ以外は業界慣習・社内標準であり、客先指定があれば客先仕様を優先する。**
> 主回路の相色（赤/白/黒）は内線規程の慣習。客先によっては全相黒＋マークチューブで区別する場合もある。

### 8-2. 端末処理の標準

| 回路種別 | 圧着端子種別 | 備考 |
|---|---|---|
| 動力回路・交流制御回路 | 丸型圧着端子 | 5.5sq超は絶縁キャップ付き、5.5sq以下はマークチューブ |
| 直流制御回路・その他制御 | Y型（先割れ）圧着端子 | |
| フェルール（棒端子） | スプリング端子台・PLC I/O端子に使用 | Wago/Phoenix Contact等のスプリング端子に対応 |

> 絶縁キャップは5.5sq超の主回路丸型端子に使用するもので、民生品の「絶縁キャップ一体型圧着端子」とは別物。
> 電線の両端には回路図の線番号をマークチューブ（指定色）で取り付ける。

### 8-2. CE対応・IEC 60204-1 準拠の場合

CE対応（機械指令）の場合は IEC 60204-1「機械の電気装置」の配線色規定が適用される。

| 回路種別 | 規定色 |
|---|---|
| 主回路（AC） | 黒（Black） |
| 制御回路（AC） | 赤（Red） |
| 制御回路（DC） | 青（Blue） |
| 接地・PE線 | 緑/黄（Green-Yellow） |
| 中性線（N） | 水色（Light blue） |

> Agent04（安全設計）がCE対応を確認した場合、本テーブルを適用する。

### 8-3. Agent03での確認事項

- `spec_summary.md` に客先指定配線色がある場合 → 客先仕様を適用
- CE対応案件（`safety_design.md` に記載） → IEC 60204-1 を適用
- 上記いずれもない場合 → 8-1の社内デフォルトを適用
- 配線色は `parts_selection.md` の備考または別途「電線仕様表」として記録する

## 10. 製作仕様の客先確認項目

盤メーカーが製作仕様書で客先に確認・承認を取る項目。Agent01の仕様解析時に確認する。

| 項目 | FA盤での一般的な標準 | 客先確認の要否 |
|---|---|---|
| 電線色 | 主回路黒・AC制御黄・DC制御青/橙・PE緑黄 | 要確認（客先指定が多い） |
| 圧着端子種別 | ねじ式端子台→裸圧着端子（丸/先割れ）、スプリング端子台→フェルール | 要確認（丸端子のみ指定も多い） |
| 端子処理（フェルール） | Wago・フェニックスコンタクト等のスプリング端子台・PLC I/O端子ではフェルール処理が一般的 | 端子台の種別による |
| 電線サイズ | 許容電流・電圧降下計算値に基づく最小サイズ | 要確認（統一サイズ指定の場合あり） |
| 電線メーカー | 指定なし（富士電線・住友電工等） | 客先指定がある場合のみ |
| マーキング方法 | チューブ印字（熱収縮チューブ） | 要確認 |
| 配線ダクト | 使用・サイズは設計による | 要確認 |

> **フェルール（棒端子）はスプリング端子台・PLC I/O端子との組み合わせでFA装置でも広く使われる。**
> ねじ式端子台（三菱TB・富士UK等）には裸圧着端子（丸/先割れ）が一般的。端子台の種別に合わせて選択する。
> 絶縁キャップ一体型圧着端子はFA装置・産業機械では一般的でない（自動車ハーネス・民生品向け）。

## 10. よくある判断ケース

| ケース | 対応 |
|---|---|
| インバータ二次側（インバータ〜モータ間） | インバータ出力電流 × 1.1〜1.25 を I_L とする |
| スターデルタ始動 | 始動時電流ではなく定格電流で選定（保護はブレーカが担当） |
| 盤内制御配線（KIV） | 束線本数が多くなりやすいため束線補正を必ず適用 |
| 線路長が不明 | 電圧降下計算を保留・ユーザー確認事項に記載する |
