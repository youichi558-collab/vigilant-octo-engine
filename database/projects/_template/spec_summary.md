## 🔴 案件初期必須確認項目

> 以下は案件受付時に最初に確認する項目。後から変わると手戻りが大きい。
> 「未確認」の場合も空欄にせず明示すること。

| No | 確認項目 | 内容 | 確度 |
|---|---|---|---|
| I-1 | 装置概要・用途 | 何の装置か（コンベア・工作機械・搬送装置等） | 確定/推定/未確認 |
| I-2 | モーター種別 | 誘導モーター / サーボモーター / ステッピングモーター / その他 | 確定/推定/未確認 |
| I-3 | 制御電源電圧 | AC200V / AC100V / DC24V / 客先指定なし | 確定/推定/未確認 |
| I-4 | 制御盤の製作・組立の有無 | あり / なし（部品選定のみ）/ 既設盤改造 | 確定/推定/未確認 |
| I-5 | 非常停止の要否 | 要 / 不要 / 未確認 | 確定/推定/未確認 |
| I-6 | 危険エリアへの立ち入り有無 | あり / なし / 未確認 | 確定/推定/未確認 |
| I-7 | CE対応・輸出要否 | 要 / 不要 / 未確認 | 確定/推定/未確認 |
| I-8 | 安全カテゴリ・PLの客先指定 | 指定あり（値:　）/ 指定なし / 未確認 | 確定/推定/未確認 |

> I-5〜I-8の回答を元にAgent00がAgent04（安全設計）の起動要否を判断する。
> I-4が「あり」の場合、Agent00がAgent05（盤製作仕様）を起動する。
