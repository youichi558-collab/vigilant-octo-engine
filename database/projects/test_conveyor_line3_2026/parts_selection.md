#### 選定根拠

**富士電機(最有力候補)**
- 参照カタログ: `電磁接触器、開閉器.pdf` p.27-31
- 適用した選定基準: 各種負荷への適用 — モータ負荷(p.27-31)
- 11kW / AC200V / IE3相当 / 直入れ始動 / AC-3使用カテゴリ → SW50X(50Aフレーム)
- サーマルリレー: TR65X、整定コード044(44〜54A)
  - ※ 全負荷電流未確定のため、銘板確認後に整定コードを再確定すること

**三菱電機(代替候補)**
- 参照カタログ: `電磁開閉器.pdf` p.46-50(2.11 モータ負荷への適用)
- 適用した選定基準: SF-PR形高効率モータ選定表(p.50)
- 11kW / AC200V / IE3 / 直入れ → MSO-T50 + TH-T50KP(34〜50A整定)
- 三菱電機公式選定ツール(配電制御器選定スケール)での確認結果と一致

- 全負荷電流が40〜43Aの場合 → コード044(44-54A)は範囲外となる可能性あり
- 銘板確認後、以下を目安に再選定すること:
  - 全負荷電流≦43A → TR65X コード040(38-50A)等を再検討
  - 全負荷電流≧44A → コード044(44-54A)で確定
- モータ効率クラスにより全負荷電流は変動するため、**必ず銘板値で確定すること**
