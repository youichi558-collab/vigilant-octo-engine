# Agent03テスト結果(2026-06-13実施)

## テスト条件
- モータ容量: 11kW / 三相200V / 標準直入れ始動用電磁開閉器

## 選定結果
| メーカー | 電磁開閉器 | サーマルリレー |
|---|---|---|
| 富士電機(本命) | SW50X | TR65X(コード044: 整定44-54A) |
| 三菱電機(代替) | MSO-T50 | TH-T50KP(34-50A整定) |

## 検証結論

- **selection_guide_ref の粒度(1カタログ1行・章範囲のみ)で実用上問題なし**
  notes列の表名・ページ情報により、40MB超のPDFでも目的の選定表に迷わず到達できた。

- 三菱電機の公式選定ツール(配電制御器選定スケール:
  https://www.mitsubishielectric.co.jp/fa/products/lvd/lvsw/ex/select/scale/index.html )
  で同条件(11kW/200-220V)を確認したところ、**MSO-T50 / TH-T50(42A整定)が公式結果と一致**。
  Agent03の選定は妥当と判断。

  - 当初「p.48標準モータ表(ヒータ呼びベース)をデフォルトにすべき」という懸念があったが、
    これは誤りだった。**現行の標準=IE3高効率モータ(SF-PR)前提のため、
    p.50「SF-PR形高効率モータ選定表」がデフォルトで正しい**。
    選定ルールへの追記は不要、現状維持でクローズ。

- 富士電機側について、富士の選定アプリ(MSスケール:
  https://f-net.fujielectric.co.jp/Catalog/FCS_appli/MSSCALE/MSSCALE.html )
  は**新SC・NEO SCシリーズ向け(SC-NEXT未対応)**のため、型式名の直接比較はできなかった。

  - 同条件(11kW/200V、全負荷電流40A)でMSスケールはヒートエレメント34-50Aを返した。
  - SC-NEXT側でAgent03が選んだTR65X(44-54A)は40Aを範囲に含まない。
  - これは選定ミスではなく、「11kW」という容量表記だけではモータの効率クラス等により
    全負荷電流(40A〜50A)が変動するため。**最終的なヒータエレメントコードは
    「モータ銘板の全負荷電流値」で確定する**(Agent03自身が確認事項としてこれを挙げていた)。

## Agent03からの確認事項(再掲・今後の入力項目検討に活用)

1. モータ銘板の全負荷電流値 → サーマルリレー整定電流コード確定に必須
2. 操作コイル電圧(AC200V / DC24V等)
3. インチング・高頻度開閉の有無 → 電気的耐久性要求の再確認

## 関連する未対応課題: 富士SC-NEXTの容量適用範囲

SC-NEXTカタログ(電磁接触器、開閉器.pdf)のラインアップ表(p.4-5)を確認したところ、
最大フレームは**SC65X/SW65X**で、

- 200-240V: 最大 **15kW(65A)**
- 380-440V: 最大 **30kW(65A)**

までしかカバーしていない。三菱(MS-T/Nシリーズ)はN125〜N800AB(〜200kW級)まで
1シリーズでカバーしているのに対し、富士はSC-NEXTが中・小容量帯のみで、
それを超える大容量帯は別シリーズ・別カタログになる可能性が高い。

→ **現状は仮説段階**。「部品カタログ」フォルダに大容量帯向けの富士カタログが
追加された時点で、index.csvへの新規行追加・SC-NEXT側のnotesに
「適用容量範囲(上限)」を追記することを検討する。
