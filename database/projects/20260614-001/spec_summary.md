## 確認質問一覧（客先送付用）

1. **モーター銘板値の確認**: No.1コンベアモーターの銘板に記載されている全負荷電流（A）をご確認ください。
2. **使用頻度**: 1時間あたりの起動回数の目安を教えてください（例: 2回以下 / 10回程度 / 頻繁など）。
3. **主回路電線路長**: 制御盤からモーターまでの電線の経路長（概算でも可）を教えてください。電圧降下計算に使用します。
4. **引込変圧器容量**: 接続する変圧器の容量（kVA）または短絡容量（kA）を教えてください。
5. **非常停止の要否**: 非常停止スイッチは必要ですか？
6. **設置環境**: 制御盤の設置場所の環境（屋内/屋外・粉塵の有無・油の有無など）を教えてください。
